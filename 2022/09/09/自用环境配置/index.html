git

  • 从官网下载 对应版本
  • 设置用户签名: git config –global user.name Dai-JF
  • 设置用户邮箱: git config –global user.email dai.jf@qq.com

Node

  • 从官网下载 对应版本

  • node -v 和 npm -v 验证是否安装成功

  • nodejs 安装目录下，创建 node_global 和 node_cache 两个文件夹

  • 设置全局模块的安装路径到 node_global 文件夹，

    设置缓存到 node_cache 文件夹

  • 把node_global路径加入path变量

  • npm install express -g 验证是否更改成功

  • 若报错 则需修改文件夹操作权限 nodejs文件夹->属性->安全，点击编辑，将所有权限都✔即可

  • 设置淘宝镜像： npm config set registry https://registry.npm.taobao.org/

  • 全局安装基于淘宝源的cnpm ：npm install -g cnpm –registry=https://registry.npm.taobao.org

  • cnpm -v 检验

yarn（基于node）

  • npm install yarn -g

  • nodejs 安装目录下，创建 yarn_global 和 yarn_cache 两个文件夹

  • 修改全局目录s

  • 把yarn_global路径加入path变量

kafka

  • 下载
  • 修改zookeeper.properties： dataDir=D:/Environment/kafka_2.13-3.2.1/zookeeper
  • 修改server.properties：log.dirs=D:/Environment/kafka_2.13-3.2.1/kafka-logs

elasticsearch

  • 下载 elasticsearc以及对应 elasticsearch-analysis-ik 插件

  • 修改elasticsearch.yml

  • 加入path变量

  • 运行bat
